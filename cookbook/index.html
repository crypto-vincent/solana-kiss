solana-kiss cookbook

No bloat. No dependencies. Full-featured Solana.

Install Connect Keypair Balance PDA Transaction IDL
01

Install

One package. Zero peer dependencies.

$ npm install solana-kiss
02

Connect to Solana

Use a cluster moniker or any custom RPC URL. Built-in monikers: mainnet, devnet, testnet.

TypeScript
import { rpcHttpFromUrl, urlRpcPublicMainnet } from "solana-kiss";

// using a built-in public endpoint
const rpc = rpcHttpFromUrl(urlRpcPublicMainnet);

// or bring your own
const rpc = rpcHttpFromUrl("https://my-rpc.example.com");
03

Generate or Import a Keypair

signerGenerate creates a fresh Ed25519 keypair. signerFromSecret restores one from a 64-byte secret.

TypeScript
import { signerFromSecret, signerGenerate } from "solana-kiss";

// generate a brand-new keypair
const signer = await signerGenerate();
console.log(signer.address); // base58 public key

// restore from a 64-byte secret key
const secretBytes = new Uint8Array([/* 64 bytes */]);
const signer = await signerFromSecret(secretBytes);
04

Get Account Balance

Lamports are bigint. lamportsToSol converts to a human-readable SOL string.

TypeScript
import {
  lamportsToSol,
  pubkeyFromBase58,
  rpcHttpFromUrl,
  rpcHttpGetAccountLamports,
  urlRpcPublicMainnet,
} from "solana-kiss";

const rpc = rpcHttpFromUrl(urlRpcPublicMainnet);
const address = pubkeyFromBase58("YourAddressHere...");

const { lamports } = await rpcHttpGetAccountLamports(rpc, address);
console.log(lamportsToSol(lamports)); // e.g. "1.5"
05

Derive a PDA

Pass the program address and an array of seed byte buffers. pubkeyFindPdaAddress returns the off-curve address.

TypeScript
import {
  pubkeyFindPdaAddress,
  pubkeyFromBase58,
  utf8Encode,
} from "solana-kiss";

const program = pubkeyFromBase58("TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA");
const wallet  = pubkeyFromBase58("YourWalletAddressHere...");

const pda = pubkeyFindPdaAddress(program, [
  utf8Encode("metadata"),
  wallet,
]);
06

Build & Send a Transaction

Compile, sign and broadcast in three straightforward steps. Await confirmation with the built-in polling helper.

TypeScript
import {
  rpcHttpFromUrl,
  rpcHttpGetLatestBlockHash,
  rpcHttpSendTransaction,
  rpcHttpWaitForTransaction,
  signerFromSecret,
  timeoutMs,
  transactionCompileAndSign,
  urlRpcPublicDevnet,
} from "solana-kiss";

const rpc    = rpcHttpFromUrl(urlRpcPublicDevnet);
const signer = await signerFromSecret(secretBytes);

// 1. fetch a recent block hash
const { blockHash } = await rpcHttpGetLatestBlockHash(rpc);

// 2. compile + sign
const packet = await transactionCompileAndSign([signer], {
  payerAddress: signer.address,
  recentBlockHash: blockHash,
  instructionsRequests: [myInstruction],
});

// 3. send and wait
const { transactionHandle } = await rpcHttpSendTransaction(rpc, packet);
const { executionReport } = await rpcHttpWaitForTransaction(
  rpc,
  transactionHandle,
  async () => { await timeoutMs(2000); return true; },
);
console.log(executionReport.transactionError); // null on success
07

Decode an Account via IDL

Solana is the high-level entry point. It auto-fetches the on-chain IDL and decodes account state in one call.

TypeScript
import { pubkeyFromBase58, Solana } from "solana-kiss";

const solana  = new Solana("mainnet");
const address = pubkeyFromBase58("YourAccountAddressHere...");

const { accountState } =
  await solana.getAndInferAndDecodeAccount(address);

console.log(accountState); // fully typed, decoded JSON
