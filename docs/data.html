Pubkey

A Pubkey is a branded string — the Base58 representation of a 32-byte Ed25519 public key. Branding prevents accidentally passing a raw string where a public key is expected.

Creating Pubkeys

import {
  pubkeyFromBase58,
  pubkeyFromBytes,
  pubkeyToBytes,
  pubkeyToBase58,
  pubkeyDefault,
} from "solana-kiss";

// From a Base58 string
const pk = pubkeyFromBase58("So11111111111111111111111111111111111111112");

// From raw bytes
const bytes = new Uint8Array(32);
bytes[31] = 1;
const pk2 = pubkeyFromBytes(bytes);

// Convert back to bytes
const rawBytes = pubkeyToBytes(pk);

// The "zero" / default public key (all bytes are 0)
console.log(pubkeyDefault); // "11111111111111111111111111111111"

PDA Derivation

Derive a Program Derived Address (PDA) from raw seeds without needing the high-level Solana class. 

import {
  pubkeyFindPdaAddress,
  pubkeyFromSeed,
  utf8Encode,
  pubkeyToBytes,
} from "solana-kiss";

const programAddress = pubkeyFromBase58("myProgram...");

// Find PDA from arbitrary seeds
const { address: pdaAddress, bump } = pubkeyFindPdaAddress(
  [
    utf8Encode("user"),          // string seed
    pubkeyToBytes(userAddress),  // pubkey seed
  ],
  programAddress
);

console.log("PDA:", pdaAddress, "bump:", bump);

// Deterministic public key from a seed + program (no bump search)
const derived = pubkeyFromSeed(utf8Encode("config"), programAddress);
ℹ️ Info
pubkeyFindPdaAddress runs the bump search (0–255) in pure TypeScript — no native module required. It uses the built-in SHA-256 implementation.

Signer

A Signer wraps an Ed25519 keypair. It exposes an address (public key) and a sign method. Everything is built in — no native add-ons or third-party libs. 

import { signerGenerate, signerFromSecret } from "solana-kiss";

// ── Generate a random keypair ──────────────────────────────────────
const keypair = await signerGenerate();
console.log("Address:", keypair.address);

// Sign a raw message (Uint8Array)
const message = new TextEncoder().encode("hello solana");
const sig = await keypair.sign(message);

// ── Load a keypair from a 64-byte secret ──────────────────────────
// Format: first 32 bytes = secret scalar, last 32 bytes = public key
// (compatible with Solana CLI `solana-keygen new` output)
const secretBytes = new Uint8Array(64); // load from secure storage
const loaded = await signerFromSecret(secretBytes);

// If you only have the 32-byte seed (not the full 64-byte secret):
const seed = new Uint8Array(32);
const fromSeed = await signerFromSecret(seed, { isSeed: true });
⚠️ Warning
Never hardcode secrets in source code. Load them from environment variables or a secure key store.

Wallet (Browser)

solana-kiss implements the Wallet Standard — a browser-native interface for discovering and interacting with wallet extensions (Phantom, Backpack, Solflare, …).

Discover Wallet Providers

import { walletProviders } from "solana-kiss";

// walletProviders is an RxObservable — subscribe to get live updates
const unsubscribe = walletProviders.subscribe((providers) => {
  providers.forEach((p) => {
    console.log(p.name, p.icon); // "Phantom", "data:image/png;base64,..."
  });
});

// Later, stop listening
unsubscribe();

Connect and Sign

import { walletProviders } from "solana-kiss";

walletProviders.subscribe(async (providers) => {
  const phantom = providers.find((p) => p.name === "Phantom");
  if (!phantom) return;

  // Connect (shows wallet popup)
  const accounts = await phantom.connect();
  const account = accounts[0];
  console.log("Connected:", account.address);

  // Sign a raw message
  const msg = new TextEncoder().encode("Sign this message");
  const sig = await account.signMessage(msg);

  // Sign a transaction packet (returns updated packet with signature)
  const signed = await account.signTransaction(unsignedPacket);

  // Disconnect
  await phantom.disconnect();
});

Silent Reconnect

// Try to reconnect without showing a popup
const accounts = await provider.connect({ silent: true });

Transaction

The transaction types represent the different stages of the transaction life-cycle: request → compiled message → signed packet → on-chain handle.

Types at a Glance

TypeDescription
TransactionRequestUnsigned inputs: payer, block hash, and instructions.
TransactionMessageCompiled transaction body (branded Uint8Array).
TransactionPacketFully signed wire-format bytes (branded Uint8Array).
TransactionHandleOn-chain signature reference (branded Base58 string).
TransactionAddressLookupTableAn ALT reference + the subset of addresses used.
TransactionProcessorFunction (packet) => Promise<packet> for post-processing/co-signing.

Compile and Sign Manually

Usually you'll use the high-level Solana class. But if you need full control, you can compile and sign a transaction yourself. 

import {
  transactionCompileAndSign,
  transactionHandleFromBase58,
  transactionHandleToBase58,
  transactionPacketFromBytes,
  transactionPacketToBytes,
} from "solana-kiss";

// Build a TransactionRequest manually
const txRequest = {
  payerAddress: payer.address,
  recentBlockHash: blockHash,
  instructionsRequests: [instructionRequest1, instructionRequest2],
};

// Compile + sign
const txPacket = await transactionCompileAndSign(
  [payer],         // signers
  txRequest,
  []               // no lookup tables
);

// Serialize for transport
const wireBytes = transactionPacketToBytes(txPacket);
const packet2   = transactionPacketFromBytes(wireBytes); // round-trip

// Work with handles (signatures)
const handle = transactionHandleFromBase58("5xA2...");
console.log(transactionHandleToBase58(handle));

Instruction

An InstructionRequest is the raw wire representation of a Solana instruction: program address, a list of account inputs, and encoded data bytes. 

import type {
  InstructionRequest,
  InstructionInput,
} from "solana-kiss";

// Build an instruction manually (without IDL)
const ix: InstructionRequest = {
  programAddress: splSystemProgramAddress,
  instructionInputs: [
    { address: payer.address, signer: true,  writable: true  }, // from
    { address: recipient,     signer: false, writable: true  }, // to
  ],
  instructionData: new Uint8Array([2, 0, 0, 0, /* transfer discriminator */ ...lamportBytes]),
};

Encoding

solana-kiss ships built-in, dependency-free encoding utilities.

Base58

import { base58Encode, base58Decode, base58BytesLength } from "solana-kiss";

const bytes  = new Uint8Array([0, 1, 2, 255]);
const str    = base58Encode(bytes);   // "12CAS"
const back   = base58Decode(str);     // Uint8Array [0, 1, 2, 255]
const len    = base58BytesLength(str);  // 4

Base64

import { base64Encode, base64Decode } from "solana-kiss";

const bytes = new Uint8Array([72, 101, 108, 108, 111]);
const str   = base64Encode(bytes);  // "SGVsbG8="
const back  = base64Decode(str);    // Uint8Array [72, 101, 108, 108, 111]

Base16 (Hex)

import { base16Encode, base16Decode } from "solana-kiss";

const bytes = new Uint8Array([0xde, 0xad, 0xbe, 0xef]);
const hex   = base16Encode(bytes);  // "deadbeef"
const back  = base16Decode(hex);    // Uint8Array [222, 173, 190, 239]

UTF-8

import { utf8Encode, utf8Decode } from "solana-kiss";

const bytes = utf8Encode("hello");    // Uint8Array [104, 101, 108, 108, 111]
const str   = utf8Decode(bytes);      // "hello"

Cryptography

SHA-256

Pure-TypeScript SHA-256 (FIPS 180-4). No Web Crypto dependency, works in any environment. 

import { sha256Hash } from "solana-kiss";

// Hash one or more byte blobs — they are fed in order
const digest = sha256Hash([
  new TextEncoder().encode("hello"),
  new TextEncoder().encode(" world"),
]);
console.log(digest); // 32-byte Uint8Array

Signature

A Signature is a branded 64-byte Uint8Array representing an Ed25519 signature. 

import { signatureFromBytes, signatureToBytes } from "solana-kiss";

const raw  = new Uint8Array(64).fill(1);
const sig  = signatureFromBytes(raw);   // throws if not 64 bytes
const back = signatureToBytes(sig);     // Uint8Array (64 bytes)

Primitives

Lamports

import {
  approximateSolsForLamports,
  approximateLamportsForSols,
  lamportsRentExemptionMinimumForSpace,
  lamportsFeePerBytePerYear,
  lamportsFeePerSignature,
} from "solana-kiss";

// Convert between SOL and lamports
const sols = approximateSolsForLamports(1_000_000_000n); // 1.0
const lamps = approximateLamportsForSols(0.5);           // 500000000n

// Estimate rent-exemption minimum for an account
const rentMin = lamportsRentExemptionMinimumForSpace(165); // ~2039280n

// Constants
console.log(lamportsFeePerSignature); // 5000n
console.log(lamportsFeePerBytePerYear); // 3480n

BlockHash and BlockSlot

import {
  blockHashFromBase58,
  blockHashFromBytes,
  blockHashToBytes,
  blockHashDefault,
  blockSlotFromNumber,
  blockSlotToNumber,

// Parse a block hash from Base58
const bh = blockHashFromBase58("EkSnNWid2cvwEVnVx9aBqawnmiCNiDgp3gUdkDPTKN1N");

// Zeroed block hash (used for simulation without verification)
console.log(blockHashDefault);

// Work with slots

console.log(blockSlotToNumber(slot)); // 123456789

Explorer URLs

Build Solana Explorer links for accounts, transactions, and blocks. 

import {
  urlExplorerAccount,
  urlExplorerTransaction,
  urlRpcPublicMainnet,
  urlRpcPublicDevnet,
} from "solana-kiss";

// Link to an account on Explorer (mainnet)
const accountUrl = urlExplorerAccount(urlRpcPublicMainnet, myAddress);
console.log(accountUrl.href);
// https://explorer.solana.com/address/myAddress

// Link to a transaction on Explorer (devnet)
const txUrl = urlExplorerTransaction(urlRpcPublicDevnet, transactionHandle);
console.log(txUrl.href);
// https://explorer.solana.com/tx/...?cluster=devnet
